# tmdb_5000_movies.csv Data Analysis
## Overview
Analysis of tmdb_5000_movies.csv demonstrating end-to-end data cleaning, feature engineering, and buisness insight.

## Key Findings
### Financial Analysis
-Budget vs Revenue correlation: 0.68 (strong positive relationship)
-Optimal Release Month: June (249802777$ average profit)

### Success Factors
-BlockBuster movies: Revenue > 3x Budget, Rating > 7/10.
-Surprise hits: Profit Margin 500%.
-Average runtime for Blockbuster movies: 123 minutes.

### Industry Patterns
-Higher budgets lead to higher revenues most of the time, but not guaranteed success.
-Ratings has weak correlation (0.25) with financial success.
-Strategic release timing impacts profitability significantly.

## Technologies
Python, Pandas, Numpy, Matplotlib, tmdb_5000_movies.csv (data set link: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
